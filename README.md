# Customer Churn Prediction

## Project Overview
This project predicts customer churn using Machine Learning models such as:

- Logistic Regression
- Random Forest
- Gradient Boosting

The model analyzes customer behavior and predicts whether a customer is likely to churn.

## Dataset
Dataset used:
customer_churn_data.csv

Features include:
- Gender
- Senior Citizen
- Tenure
- Monthly Charges
- Contract Type
- Internet Service
- Payment Method
- Total Charges

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Jupyter Notebook

## Models Used
1. Logistic Regression
2. Random Forest
3. Gradient Boosting

## Best Model
Best model selected based on:
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC
